## 功能特性

- [x] 支持文字/图片/音频/视频投稿
- [x] 支持多图投稿
- [x] 支持匿名投稿
- [x] 支持直接发布投稿
- [x] 支持编辑标签
- [x] 支持过滤标签
- [x] 简单用户管理
  - [x] 设置用户组
  - [x] 封禁/解封
  - [x] 封禁记录查询
  - [x] 查看用户投稿统计
  - [x] 匿名回复
  - [x] 检索用户
- [x] 审核超时自动拒绝
- [x] 投稿数量限制
- [x] 获取随机稿件
- [x] 自定义稿件标签
- [x] 自定义拒绝理由
- [x] IPC 接口 (使用命令 /token 获取凭据)
- [x] 实装用户等级系统
- [ ] 稿件检索
